# Java Data Structures

This repository is to store all my exercise files with data structures using Java.

Here are the data structures covered and operations

1. Arrays
    1. Check if the sum is in a Sorted Array
    2. Rotate an Array by 'd' digits
2. Linked List
    1. Add element to Linked List
    2. Delete an element from Linked List
    3. Find if the Linked List is circular
3. Stack (coming soon)
4. Queue (coming soon)
5. Trees (coming soon)

## How to use it

1. Clone the repo
2. Run the maven build - mvn clean install
